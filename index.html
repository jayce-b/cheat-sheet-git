My Cheat Sheet : Git Commands

Créer un Dépôt sur Github

  1. aller sur le site GitHub
  2. créer un compte :
    • choisir un nom d'utilisateur
    • entrer son adresse e-mail
    • créer un mot de passe
    • s'enregistrer
  3. créer un dépôt
    • entrer un nom de dépôt
    • entrer une description du contenu du dépôt
    • cocher l'option "public"
    • initialiser le dépôt avec le fichier "README.md" pour détailler le contenu du dépôt
    • cliquer sur créer un dépôt
