My Cheat Sheet : Git Commands

Créer un Dépôt sur Github

  1. aller sur le site GitHub
  2. créer un compte
    • choisir un nom d'utilisateur
    • entrer son adresse e-mail
    • créer un mot de passe
    • s'enregistrer
  3. créer un dépôt
    • entrer un nom de dépôt
    • entrer une description du contenu du dépôt
    • cocher l'option "public"
    • initialiser le dépôt avec le fichier "README.md" pour détailler le contenu du dépôt
    • cliquer sur créer un dépôt

Les Commandes de Base

git clone [URL] récupère le dépôt à partir d'une adresse URL

git add [fichier] marque le fichier à consigner

git add * marque tous les fichiers à consigner

git commit -m consigne les fichiers marqués dans le dépôt

git status affiche l'état du dépôt local

git push origin master envoie les fichiers modifiés sur le dépôt principal

Encore plus sur Git

git reset --hard HEAD~1 supprime la dernière modification consignée (avant d'avoir fait un push)

git log affiche l'historique des commandes (q pour sortir)

git config --list affiche la liste des paramètres de git

git config --global [username] enregistre le paramètre "nom utilisateur" du dépôt

git config --global [useremail] enregistre l'email de l'utilisateur du dépôt
